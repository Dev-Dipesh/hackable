## Requirements
- NodeJS
- MongoDB
- Redis
- Docker and Docker Compose *optional

## Getting Started
1. git clone git@github.com:uptownhr/hackathon-starter-lite
2. npm install
3. npm run dev

## Getting Started using Docker
1. git clone git@github.com:uptownhr/hackathon-starter-lite
2. docker-compose up
